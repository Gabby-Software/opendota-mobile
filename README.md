Beta is Open
----
How to get invited:
1. Head over to our [discord](https://discord.gg/0o5SQGbXuWCNDcaF).
2. Post a message on the mobile channel or send a private message to Proyoyo with your:
    - Email address
    - Device type (e.g. iPhone 8, Samsung Galaxy S8+, etc)
3. You will receive an email from Mobile Center Team (Android) or TestFlight (iOS) with the invitation and download instruction.

Please submit bug reports to Github issue page. If there's any question, feel free to post in our discord mobile channel/send a private message to Proyoyo in discord. 

Mobile Center Download Instruction
----
1. Open the invitation email from Mobile Center Team and click "See Details".
<img src="https://i.imgur.com/WiUFiFA.png" height="640" width="360"/>

2. Create a new account to Mobile Center or login using existing credentials.
<img src="https://i.imgur.com/xmFQtFf.png" height="640" width="360"/>

3. Click Download. **WARNING! Do not click on "Latest version is available here" as it does not always get the latest available version.**
<img src="https://i.imgur.com/ejopAVs.png" height="640" width="360"/>

4. Install the downloaded apk.

TestFlight Download Instruction
----
1. Open the invitation email from TestFlight and click "View in TestFlight".
<img src="https://i.imgur.com/AEUduMb.png" height="239" width="318"/>

2. Click "Get TestFlight from the App Store". Download TestFlight and open it.
<img src="https://i.imgur.com/Pi3LDOu.png" height="222" width="339"/>

3. Click on "Redeem" and enter the code in step 2.

Tech Stack
----
* Cross-platform development: React Native

Quick Start (React Native)
----
* Follow the official [React Native guide](https://facebook.github.io/react-native/docs/getting-started.html) to setup your environment.
* Clone the repo
* `cd` into local folder and do a `yarn install` to install dependencies (or use `npm install`)
* run `react-native run-ios` or `react-native run-android`
